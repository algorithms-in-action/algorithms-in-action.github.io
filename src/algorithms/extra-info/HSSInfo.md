## Extra Info

Geeks for Geeks Link: [**Horspool-Boyer-Moore String Pattern Search**][G4GLink]


[G4GLink]: https://www.geeksforgeeks.org/boyer-moore-algorithm-for-pattern-searching/

Horspool's algorithm (also called Boyer-Moore-Horspool) is a simplification of the Boyer-Moore algorithm that is much
simpler to code and has good behavior.  The original Boyer-Moore algorithm has a "Good-Prefix" table,
as well as the "Bad Character" (Shift) table in the Horspool simplification.

Look up the original Boyer-Moore algorithm and try some test cases to see how it differs from the Horspool algorithm shown in AIA.
